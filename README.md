# 職務経歴書ドラフト作成ツール

自己分析の質問にいくつか答えるだけで、職務経歴書の下書き文章を自動でつくるWebアプリです。

公開URL: https://yuuki9608.github.io/career-draft-app/

## 対象ユーザー

求職者・学生本人が、自分自身の自己分析のために使うことを想定しています。転職活動・就職活動の準備として、職務経歴書の下書きをつくりたい人向けです。

## 使い方

1. 上記の公開URL、またはこのフォルダの `index.html` をダブルクリックしてブラウザ(Google Chrome / Microsoft Edge を推奨)で開く
2. 「はじめる」から、基本情報 → 職歴・経験 → 強み・スキル → PCスキル → 自己PRのもと、の順に入力する(職歴が無くても0件のまま進められます)
3. 最後の画面で「下書きを生成する」を押すと、職務経歴書の下書き文章が表示される
4. 内容は画面上で自由に書き直せる。「コピー」「テキストでダウンロード」「印刷してPDF保存」で好きな形式で持ち出せる

入力内容は自動でブラウザに保存されるため、途中で閉じても「前回の続きから」で再開できます。

## 技術構成・選定理由

- **Vanilla HTML / CSS / JavaScript のみ(フレームワーク・ビルドツール不使用)**。GitHub Pagesにそのまま公開でき、`index.html` をダブルクリックしてブラウザで直接開くだけでも動作するようにしています。
- ES module や `fetch()` によるローカルファイル読み込みは使っていません。これは `file://` で開いたときにブラウザのセキュリティ制限で動かなくなるのを避けるためです。
- CDN(フレームワーク・Webフォント含む)は一切使用していません。システムフォントとインラインSVGのみで構成し、完全にオフラインでも壊れないようにしています。
- 下書きの生成は、あらかじめ用意した文章パターンに入力内容を当てはめる**テンプレート方式**です。AI(生成AI API)は使用していません。ネット接続不要・追加費用不要・待ち時間なしで動作します。

## プライバシーについて

入力した内容はすべて**お使いのブラウザの中(localStorage)だけ**に保存され、外部のサーバーには一切送信されません。ブラウザの開発者ツールの「Network」タブを開いた状態で操作しても、通信が発生しないことを確認できます。

「すべてのデータを消す」から、保存内容をいつでも完全に削除できます。
